== Description ==

Bloquez n’importe quelle adresse IP sur votre site (back et front) et soyez notifié par mail et sms (pour les abonnés Free Mobile) lorsqu’une personne avec avec l’adresse IP blackilisté essaye de rentrer sur votre site.

== Installation ==

1. Uploadez `ip—blocker-wp-par-par-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’IP Blocker WP’ depuis le menu « Extensions » de Wordpress
